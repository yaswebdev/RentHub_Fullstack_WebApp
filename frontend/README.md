## ✨ Points Forts de la Version Premium

### 🌓 Mode Sombre Immersif
Une expérience visuelle entièrement repensée pour le confort nocturne. Le mode sombre utilise une palette de couleurs **Indigo & Slate** profonde avec des effets de verre dépoli (glassmorphism) sur tous les composants.

### 🎨 Design System Moderne
- **Nuances Indigo/Violet** pour une identité visuelle premium.
- **Micro-animations** fluides avec Framer Motion (sur les boutons, les cartes et les modals).
- **Glassmorphism** : Utilisation de bordures subtiles et de flous d'arrière-plan pour une profondeur maximale.

### 📱 Fonctionnalités Clés
- **Recherche Avancée** : Filtres intelligents, curseur de prix dynamique et recherche par ville.
- **Tableau de Bord Voyageur & Hôte** : Statistiques détaillées, gestion des annonces et suivi des revenus.
- **Messagerie iMessage-style** : Interface de chat ultra-réactive avec réponses automatisées.
- **Détails de Propriété Enrichis** : Galerie photo immersive, informations hôte détaillées et carte interactive intégrée.

## 🛠️ Stack Technique

- **Frontend**: React 18
- **Styling**: Tailwind CSS (Vanilla)
- **Animations**: Framer Motion & Lucide React
- **Gestion d'état**: React Context API
- **Maps**: Intégration Leaflet / OpenStreetMap
- **Data**: Mock API & LocalStorage (Simulateur Firebase local)

## 🚀 Installation Locale

**Prérequis :** Node.js installée sur votre machine.

1.  **Cloner le dépôt** :
    ```bash
    git clone https://github.com/mamadox001/RenHub---Google-Ai-Studio.git
    cd RenHub---Google-Ai-Studio
    ```

2.  **Installer les dépendances** :
    ```bash
    npm install
    ```

3.  **Configurer les variables d'environnement** :
    Copiez le fichier `.env.example` en `.env` et ajoutez vos clés si nécessaire.
    ```bash
    cp .env.example .env
    ```

4.  **Lancer l'application** :
    ```bash
    npm run dev
    ```
